# Chronik Documentation

Chronik is a fast and reliable indexer built into the Bitcoin ABC node software.

## Getting started

- If you want to set up your own Chronik instance, [start here](/chronik/setup/setup-chronik/). **Note:** This requires a decent server and may take days to sync. Recommended for advanced users.
- If you want to use Chronik in your JavaScript or TypeScript app, [start here](/chronik/chronik-client/install/).

If anything is unclear or you spot a problem in this documentation, the Bitcoin ABC team is available via [t.me/eCashDevelopment](https://t.me/eCashDevelopment).
